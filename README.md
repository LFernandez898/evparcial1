Descripción del Proyecto
Este proyecto consiste en el desarrollo de un flujo profesional de Ciencia de Datos centrado en la limpieza y preparación de un dataset "crudo" de FIFA 21. El objetivo es transformar datos inconsistentes en un conjunto de información listo para el modelado predictivo, aplicando técnicas avanzadas de ingeniería de características y automatización mediante pipelines.


 Estructura del Repositorio
Siguiendo los estándares de reproducibilidad exigidos, el repositorio se organiza de la siguiente manera:


/notebooks: Contiene el archivo .ipynb con el análisis detallado y el código de transformación.

/data: Incluye el dataset original (raw) y el dataset final procesado (processed).

/src: Funciones modulares y scripts auxiliares para la limpieza de datos.

/outputs: Visualizaciones y reportes generados durante el análisis exploratorio.

Justificación del Entorno
Hemos seleccionado Google Colab como nuestra estación de trabajo principal:

Colaboración Remota: Facilita el trabajo simultáneo de los tres integrantes.

Integración con GitHub: Permite mantener un historial de versiones (commits) constante, cumpliendo con el requisito de versionado real.

Recursos en la Nube: Garantiza la reproducibilidad del código sin importar la potencia de hardware local de los integrantes.


Tecnologías Utilizadas
Python: Lenguaje principal.

Pandas & NumPy: Manipulación y limpieza vectorizada de datos.

Scikit-Learn: Implementación de Pipelines para escalado y codificación avanzada.
